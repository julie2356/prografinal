Calcular Promedio de 4 notas

Descripcion

Este código calcula el promedio de cuatro notas y muestra el resultado en una alerta.Primero, suma las notas y divide el total entre 4 para obtener el promedio, que se redondea usando Math.round(). Finalmente, muestra el resultado en una alerta. Es una forma simple de realizar cálculos y mostrar resultados en JavaScript.

Codigo Javacript


                //Calcular el promedio de 4 notas

                //Declarar variables
                let nota1 = 98;
                let nota2 = 88;
                let nota3 = 94;
                let nota4 = 90;

                //Realizamos la operacion
                promedio =Math.round((nota1+nota2+nota3+nota4)/4); 

                //Imprimir el resultado
                alert(" El resultado del promedio de 4 notas es: "+promedio)
