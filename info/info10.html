Reloj digital

Descripcion

Un reloj digital es un dispositivo que muestra la hora en formato numérico, a diferencia de un reloj analógico que utiliza manecillas y una esfera. Los relojes digitales pueden mostrar la hora, minutos y segundos de manera clara y precisa, y a menudo incluyen funciones adicionales como alarmas, temporizadores y cronómetros.

Codigo Javacript


                function updateClock() {
                    const now = new Date();
                    const hours = String(now.getHours()).padStart(2, '0');
                    const minutes = String(now.getMinutes()).padStart(2, '0');
                    const seconds = String(now.getSeconds()).padStart(2, '0');
                    const timeString = `${hours}:${minutes}:${seconds}`;
                    document.getElementById('clock').textContent = timeString;
                }
                
                setInterval(updateClock, 1000);
                updateClock();
