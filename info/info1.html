Calculadora Basica

Descripcion

Esta calculadora web es una herramienta simple y eficaz diseñada para realizar operaciones aritméticas básicas.El código define funciones en JavaScript para realizar operaciones matemáticas básicas: suma, resta, multiplicación, división y cálculo del residuo. Cada función toma dos números de campos de entrada en un formulario, realiza la operación correspondiente y muestra el resultado en otro campo de entrada. Es útil para crear una calculadora simple en una página web.

Codigo Javacript


            function suma(){
                var numero1 = parseFloat(document.getElementById('numero1').value);
                var numero2 = parseFloat(document.getElementById('numero2').value);
                var resultado = numero1 + numero2;
                
                document.getElementById('resultado').value = resultado;
             }           
                
            function resta(){
