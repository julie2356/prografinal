Area de un circulo

Descripcion

Este código calcula el área de un círculo dado su radio y muestra el resultado en la página web.Primero, se declara una variable radio con un valor de 10. Luego, se calcula el área multiplicando π (con Math.PI) por el cuadrado del radio. Finalmente, se muestra el resultado en la página web utilizando document.write().

Codigo Javacript


                //Calcular el area de un circulo

                //Declarar variables
                let radio = 10;

                //Realizar el proceso
                area = Math.PI * (radio * radio);

                //Imprimir los resultados
                document.write("El area de un circulo con un radio de ", radio , " es igual a " , area);
