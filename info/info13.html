Mayor o Menor de edad

Descripcion

En este ejercicio de JavaScript, se desarrolla una función que determina si una persona es mayor de edad o no, basándote en su edad ingresada. La mayoría de los países consideran que una persona es mayor de edad a partir de los 18 años, por lo que este ejercicio se centrará en esa edad como umbral.

Codigo Javacript


                //Definir Variable
                let edad = parseFloat(prompt("Ingrese su edad "));

                //Realizar procedimiento
                if (edad >= "18"){
                    document.write(" Usted es mayor de edad ");
                }
                else if (edad <= "17"){
                    document.write(" Usted es menor de edad ");
                }
                else {
                    document.write("Ingrese un valor");
                }
