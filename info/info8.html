Tip Calculator

Descripcion

En este ejercicio de JavaScript, desarrollar√°s una calculadora de tips que ayuda a los usuarios a calcular el monto de la propina a partir de la cuenta total de una comida o servicio.

Codigo Javacript


        document.getElementById('tip-form').addEventListener('submit', function(event){
        event.preventDefault();

        //Obten los valores ingresados por el usuario
        const billAmount =
        parseFloat(document.getElementById('bill-amount').value);
        const tipPercentage =
        parseFloat(document.getElementById('tip-percentage').value);

        //Calcular la propina 
        const tipAmount = (billAmount * tipPercentage) / 100;

        //Calcular el monto total con propina
        const totalAmount = billAmount + tipAmount;

        //Resultados 
        document.getElementById('tip-amount').textContent = `Propina: $${tipAmount.toFixed(2)}`;
        document.getElementById('total-amount').textContent = `Monto total con propina: $${totalAmount.toFixed(2)}`;


    });
