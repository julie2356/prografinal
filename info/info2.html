Calcular Perimetro de un Circulo

Descripcion

Este ejercicio tiene como objetivo calcular el perímetro de un círculo dado su radio. Se declara una variable radio con un valor de 10 y se aplica la fórmula P=2πr para obtener el perímetro. Luego, se imprime el resultado en la página web. La función Math.PI se utiliza para obtener el valor de π. Este código es útil para entender cómo aplicar fórmulas matemáticas en programación.

Codigo Javacript


                //Calcular el perimetro de un circulo

                //Declarar variables
                let radio = 10;

                //Realizar el proceso
                perimetro = 2* Math.PI * radio;

                //Imprimir los resultados
                document.write("El perimetro de un circulo con un radio de ", radio , " es de: " , perimetro);
