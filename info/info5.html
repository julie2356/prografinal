Adivinar Numero

Descripcion

El juego "Adivina el Número" es un juego interactivo donde el jugador intenta adivinar un número secreto generado aleatoriamente entre 1 y 100. El jugador tiene que introducir su suposición, y el juego le proporciona pistas sobre si el número secreto es más alto o más bajo que su intento. El objetivo es adivinar el número secreto en la menor cantidad de intentos posibles.

Codigo Javacript


                let numeroSecreto = Math.floor(Math.random()*100)+1;
                let intentos = 0;
                
                function adivinaNumero(){
                    const intento = parseInt(document.getElementById('intento').value);
                    intentos++;
                    const pista = document.getElementById('pista');
                
                    if (intento < numeroSecreto){
                        pista.textContent = 'Mas alto';
                    } else if (intento > numeroSecreto){
                        pista.textContent = 'Mas bajo'
                    } else {
                        pista.textContent = `Felicidades, adivinaste el numero en ${intentos} intentos.`;
                    }
                }
